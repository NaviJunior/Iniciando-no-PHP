# CRUD em PHP

Projeto desenvolvido para praticar operações de **CRUD (Create, Read, Update e Delete)** utilizando **PHP**, **MySQL** e **Bootstrap**.

## 🚀 Tecnologias

* PHP
* MySQL
* Bootstrap 5

Isso é uma alteração


## 📌 Funcionalidades

* Cadastrar registros
* Listar registros
* Editar registros
* Excluir registros


## 🎯 Objetivo

Este projeto foi desenvolvido com fins de estudo para praticar a integração entre PHP e MySQL, além da implementação das operações básicas de um CRUD.
